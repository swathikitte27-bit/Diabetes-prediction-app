# Diabetes Prediction Web App 🩺
Predicts diabetes using SVM model trained on Pima Indians dataset.

**Live Demo:** https://diabetes-prediction-app-cqj9jtixbooydufcjc94ke.streamlit.app/

### Features
- Takes 8 medical inputs: Pregnancies, Glucose, BP, etc.
- Real-time prediction using trained model
- Built with Streamlit

### Tech Stack
Python, Scikit-learn, Streamlit, Numpy, Pandas
